About Farmassist

Brief Description

Farmassist is a smart farming app for IoT and AI-powered plant disease detection. It is built with Flutter and uses Firebase as its backend.

Frontend:   Flutter
Languages:   Dart, MySQL
Backend:   Firebase
Dashboard:   Grafana
